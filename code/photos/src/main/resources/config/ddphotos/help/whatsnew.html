What's New?

Version 1.0.0b6 - June 21st, 2026
  • The setup wizard now shows a beta notice and asks you to accept the license terms before continuing.
  • More consistent status and error messages in the command console.
Version 1.0.0b5 - June 20th, 2026
  • Windows support, including a new wizard step to locate Git Bash.
  • Stopping the serve and run commands now reliably stops the container.
  • Add settings.debug.local.image debug option which, if set, uses local ddphotos image instead of pulling the latest official image.
Version 1.0.0b4 - June 19th, 2026
  • Add search functionality to command console. CMD-F/CTRL-F triggers it.
  • Add some more search paths for docker binary.
  • Support default_theme site setting.
  • Scroll to top in command consoles.
  • On Windows/Linux, the file picker now shows hidden files and supports creating new folders.
  • The wizard now shows the commands it runs, just like the command tabs.
  • Force-stopping a command now shows its activity in the console.
  • Various UI refinements to radio buttons, checkboxes and widget spacing/alignment.
Version 1.0.0b3 - June 15th, 2026
  • Add --aws-profile NAME parameter to deploy.
  • Make upgrade always check for new image.
Version 1.0.0b2 - June 14th, 2026
  • More robust PATH handling for finding binaries.
  • Allow rerunning install script step in wizard.
  • Have wizard always pull latest DD Photos Docker image.
Version 1.0.0b1 - June 14th, 2026
  • First beta release of DD Photos!
