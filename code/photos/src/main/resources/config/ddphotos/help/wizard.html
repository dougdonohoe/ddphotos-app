Welcome to the DD Photos Setup Wizard!

DD Photos helps you quickly create, manage, and publish photo self-hosted photo galleries. It uses the open-source ddphotos project under the hood, which itself uses Docker.

This Wizard walks through some steps to help you get started:


  1. Docker Check - Ensure Docker is installed and running and that we know the location of the docker executable.

  2. Locate Git Bash (Windows Only) - On Windows, the ddphotos script runs under Git Bash, so we need the location of your bash.exe from Git for Windows. This step is skipped on macOS and Linux.

  3. Install ddphotos script - Using Docker, download the ddphotos image and use it to install the ddphotos script, which is used to run all commands.

  4. Create a starter site - If you don't have a DD Photos site yet, run ddphotos init to create the DD Photos folders and a sample configuration so you can explore commands right away.

  5. Add DD Photos Site - Add the site you just created (or already had) and then start the normal DD Photos editor.
